1.1. python3 pip 安装 Pattern出错

pip3 install Pattern出现如下错误：

主要是这一句：

OSError: mysql_config not found

原因是centos需要安装mysql相关的一些依赖包，执行如下命令。然后就可以正常安装了。

yum install mysql-devel gcc gcc-devel python-devel
